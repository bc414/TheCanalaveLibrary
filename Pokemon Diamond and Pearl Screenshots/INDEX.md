# Diamond/Pearl Screenshot Index

Reference library for the site's design language (canvas/surface/token mapping, UI grammar).
Screenshots are from a let's play — colors here are the *as-played* palette, which is
substantially more muted than crisp tileset renders (Route201.png / Route205.png at repo
root). When sampling token values, prefer these.
