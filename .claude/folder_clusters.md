# Folder Clusters — Codebase Maintenance Reference (v4)

> **Source:** `canalave_library_unified_spec.md` §7.4 (Folder Clusters) and feature specifications throughout §5.
> **Namespace rule:** One namespace per project, regardless of folder organization (§7.4).
> **Clustering criterion:** Do these features share a Layer-1 entity cluster or a Layer-2 interface family?

> **UI columns:** Logic = `@code` behavior (parameters, services, events, state). Structure = component composition and HTML skeleton. Style = Tailwind classes, sprites, visual treatment. These form a dependency chain: Logic → Structure → Style. N/A means the layer genuinely doesn't apply. Missing means it applies but no decisions exist yet. Blocked means a prerequisite (typically design tokens for Style) hasn't been met.
